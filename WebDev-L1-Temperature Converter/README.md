# Temperature Converter

## Oasis Infobyte Internship - Web Development

### Project Overview
The Temperature Converter is a simple and responsive web application that allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin. It is built using HTML, CSS, and JavaScript with a clean and user-friendly interface.

## Features
- Convert Celsius to Fahrenheit and Kelvin
- Convert Fahrenheit to Celsius and Kelvin
- Convert Kelvin to Celsius and Fahrenheit
- Real-time temperature conversion
- Input validation for accurate results
- Responsive design for desktop and mobile devices

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure


## How to Run
1. Download or clone this repository.
2. Open the `WebDev-L1-TemperatureConverter` folder.
3. Double-click `index.html` or open it in any web browser.
4. Enter the temperature value, select the input unit and output unit, then click the convert button.
