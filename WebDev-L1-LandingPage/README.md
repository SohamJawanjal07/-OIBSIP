# Landing Page

## Oasis Infobyte Internship - Web Development Level 1

### Project Description
This project is a responsive landing page developed as part of the Oasis Infobyte Web Development Internship. The landing page is designed with a clean user interface and responsive layout to provide a seamless experience across different devices.

### Features
- Responsive design
- Modern user interface
- Navigation bar
- Hero section
- About section
- Services section
- Contact section
- Smooth scrolling (if implemented)

### Technologies Used
- HTML5
- CSS3


### Folder Structure
```
WebDev-L1-LandingPage/
│── index.html
│── style.css
└── README.md
```

### How to Run
1. Download or clone the repository.
2. Open the `WebDev-L1-LandingPage` folder.
3. Open `index.html` in any web browser.
